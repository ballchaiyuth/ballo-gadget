# BalloGadget 🎒📱

**BalloGadget** is a personal showcase project demonstrating a modern, multilingual microservices architecture. It serves as a curated management system for tech gadgets, bags, and everyday carry (EDC) loadouts.

## 🚀 The Vision

As a gadget lover and software engineer, this project bridges my passion for high-end gear (Aer, Alpaka, Mechanical Keyboards) with cutting-edge software practices.

## 🛠 Tech Stack

- **BFF (Backend for Frontend):** Java Spring Boot (Focus: TDD & Clean Architecture)
- **Gear Description Service:** Python (Focus: AI Integration with Gemini Pro)
- **Loadout Manager:** Go (Focus: High-performance SQL operations)
- **Infrastructure:** AWS (EC2, RDS, S3) managed via Terraform
- **Frontend:** Vue.js (The expert choice for a polished UI)
